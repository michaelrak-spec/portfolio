# michaelrak.org — Personal Portfolio

My personal portfolio site. I'm a People Operations and Systems leader who builds the internal software my teams need.

**🔗 Live site:** https://michaelrak.org

## About

A single-file, dependency-free portfolio built by hand — no framework and no build step. It covers the internal tools I've designed and shipped, my work history, and how to get in touch. The résumé is downloadable right from the page.

## Tech

- Hand-written **HTML, CSS, and vanilla JavaScript** — single file, no build step
- Responsive dark theme with scroll-reveal animations
- Hosted on a static host with a custom domain (`michaelrak.org`)
